
# Obstacle-Avoiding-Robot-Car

This project proposes robotic vehicle that has intelligence built in it such that it directs itself 
whenever an obstacle comes in its path

# Components

-> Arduino UNO  
-> UltraSonic Sensor  
-> DC servo motor  
-> Motor Driver(L293D)  
-> Battery  
-> Bread Board  
-> Jumper wires/Cables  
-> Wheels
